# vm-manager-backend

一个轻量级的单机qemu kvm虚拟机管理器   
A lignt qemu kvm vm manager for single machine

## 系统依赖 (System Requirement)

操作系统(OS): Ubuntu 20.04

Init: systemd

软件包依赖(software packages)：

| 软件包(package) | 版本(version) |
|:------------:|:-----------:|
|   qemu-kvm   |    >=4.2    |
|  qemu-utils  |    >=4.2    |
